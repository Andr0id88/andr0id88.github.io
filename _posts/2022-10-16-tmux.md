### Collection of sane TMUX keybinds

```bash
-----------------------------------------------------------------------
-                              Sessions                               -
-----------------------------------------------------------------------
C-c create session
d   detatch from the session
C-f find session with fzf
j   visual session\pane scroller
$  name session

<cmd> tmux list-sessions   list current open sessions
<cmd> tmux attach-session -t <session-name>   to reattach to a session
-----------------------------------------------------------------------
-                               Windows                               -
-----------------------------------------------------------------------
c   create window
X  kill window
Tab last-window
w   list windows
C-h previous-window
C-l next-window
f  find window
,  name window
-----------------------------------------------------------------------
-                                Panes                                -
-----------------------------------------------------------------------
##### Important binds #####

s    synchronize-panes <on\off>
C-j  join-pane from another window, example :4
!    move pane to a new window
x    kill pane
q    move to pane by number
o    swap panes
+    break pane into window (e.g. to select text by mouse to copy)
-    restore pane from window
z toggle pane zoom

##### Basic #####

h  move left
j  move down
k  move up
l  move right
_  vertical split
-  horizontal split
â½  space - toggle between layouts
> swap current pane with the next one
< swap current pane with the previous one

# pane resizing
H Left resize
J Down resize
K Up resize
L Right Resize
```
