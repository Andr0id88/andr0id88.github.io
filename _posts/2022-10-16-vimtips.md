#### Keybinds for vim plugins, and some tips and tricks

```bash
-----------------------------------------------------------------------
-                               FZF-VIM                               -
-----------------------------------------------------------------------
<space>  - fh              # Find recently opened file
<space>  - c               # Open all vim commands in FZF
<space>  - s               # Insert snippet from .snippy
<space>  - b               # Show vim buffers
<space>  - fl              # Find line in document
<space>  - fb              # Find line in buffers
<space>  - ft              # Find tags
<space>  - fr              # Find helptags
<space>  - f:              # Find previously run commands
<space>  - f/              # Find previously searches
<space>  - fg              # Find tracked git files
<space>  - fs              # Git diff(?)
<space>  - fc              # Find previously commits
<leader> - <space>         # Open FZF from /
<space>  - <space>         # Open FZF in current working directory

-----------------------------------------------------------------------
-                           Vim - Surround                            -
-----------------------------------------------------------------------
                      Adding `ys` (_you surround_)
        ys takes a valid vim motion or text object as first argument
-----------------------------------------------------------------------
ysiw"                      # surround inner word with "
yss]                       # surround current line with ]
ys2aw*                     # add * around 2 words
vS"                        # visually surround selected text with "
-----------------------------------------------------------------------
                    Changing `cs` (_change surround_)
  Takes 2 arguments a _target_ to replace & a _replacement_ character
-----------------------------------------------------------------------
cs])                       # change surrounding [] with ()
cst<h3>                    # change surrounding html tag with h3
-----------------------------------------------------------------------
                    Deleting `ds` (_delete surround_)
-----------------------------------------------------------------------
ds"                        # delete surrounding "
dst                        # delete surrounding tag (HTML)

-----------------------------------------------------------------------
-                    Random tips and tricks - WIP                     -
-----------------------------------------------------------------------
# Save with sudo VIM
:w !sudo -S tee %

-------------------
-  Reverse lines  -
-------------------
:%!tac

-------------------------------------
-  Get input from terminal command  -
-------------------------------------
:r!whoami

--------------------
-  Or from a file  -
--------------------
:r!/home/user/myfile
```
